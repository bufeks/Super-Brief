# リサーチ観点フレームワーク (10 領域)

`research-brief` スキルが Step 3 (計画) と Step 4 (リサーチ) で参照する観点リスト。
案件によってウェイトは変えるが、最低 6 領域はカバーする。

---

## 1. ブリーフ・デコード

| 観点 | チェックリスト |
|---|---|
| 表層課題 | ブリーフに書いてある "解いて欲しい問題" を一文で要約 |
| 真因仮説 | 表層課題の **裏にある真の課題** を 2〜3 個 |
| ターゲット定義 | デモグラ vs サイコグラ / セグメントの広さ / 一次/二次 |
| 成功指標 | 認知 / 想起 / 好意 / 購入意向 / 売上 / ロイヤルティ |
| 制約 | 予算 / 期間 / 媒体 / 表現 / 法規 |
| 違和感 | ブリーフを読んで「これ本当?」と感じた点 (3 個以上) |
| 残存する問い | クライアントに確認したいこと |

## 2. 競合状況

| 観点 | 調べ方の例 |
|---|---|
| 直接競合 | 同カテゴリー上位 3〜5 社、直近 2 年のキャンペーン棚卸し |
| 間接競合 | 同じ "時間 / 予算 / Job" を奪う他カテゴリー |
| カテゴリーリーダー | ブランド資産、定番訴求、SOV |
| 新興プレイヤー | D2C、スタートアップ、海外勢の参入 |
| 越境競合 | 隣接カテゴリーからの進出 |
| ポジショニング | 機能 / 情緒 / 価格 / 体験の 4 象限マップ |
| 共有テンプレ | 業界全体で使い古された表現 (避けるべき) |

## 3. 市場・カテゴリー

| 観点 | 一次ソース候補 |
|---|---|
| 市場規模 | 矢野経済、富士キメラ、業界団体、政府統計 (e-Stat) |
| 成長率 | 過去 3 年 CAGR、今後 3 年予測 |
| 構造変化 | チャネルシフト (EC化率)、価格帯シフト、ユーザー層変化 |
| カテゴリーの成熟度 | 導入 / 成長 / 成熟 / 衰退 |
| 規制 | 景表法 / 薬機法 / 食品表示 / 業界自主基準 |
| 季節性・周期性 | 月別販売動向 / 行事連動 |
| 上流動向 | 原材料コスト、為替、サプライチェーン |

## 4. 社会の空気 (Zeitgeist)

| 観点 | ソース候補 |
|---|---|
| 価値観の潮流 | 博報堂生活総研、電通 D-Camp、リクルートワークス、内閣府世論調査 |
| 世代別動向 | データに基づく行動・価値観 (世代論の決めつけは避ける) |
| 流行語 / バズ | 直近 6〜12 ヶ月のキーワード |
| 文化的瞬間 | カテゴリーに紐づく社会的トピック |
| 国際比較 | 海外で先行している類似動向 (日本適用性を必ず吟味) |

## 5. 生活者インサイト

| 観点 | 手法 |
|---|---|
| N1 の声 | SNS 検索、レビューサイト、5ch、note、Yahoo知恵袋 から要旨引用 |
| Job to be done | 生活者がこのカテゴリーで "雇っている" 役割 |
| Tension | 期待と現実のズレ、隠れた不満 |
| Pain / Gain | 痛み / 得たい喜び |
| 購買意思決定要因 | 価格 / 機能 / ブランド / レビュー / 口コミ / パッケージ |
| 利用シーン | いつ / 誰と / どこで / どんな気分で |
| カテゴリー語彙 | 生活者が実際に使っている言葉 (広告言葉ではなく) |

## 6. ブランド現在地

| 観点 | データ源 |
|---|---|
| 認知 / 純粋想起 / 助成想起 | 公開調査、過去 IR |
| 好意度 / イメージ連想 | 過去調査 / SNS センチメント |
| 過去キャンペーン棚卸し | 直近 3〜5 年、コア訴求の変遷 |
| 想起されている価値 | 「〇〇と言えば」のスロット |
| 現在のトーン&マナー | クリエイティブの一貫性 |
| ロイヤル層の声 | レビュー、コミュニティ |
| 離反理由 | レビュー / SNS の否定的言及 |

## 7. カルチャー文脈

| 観点 | 探し方 |
|---|---|
| 関連サブカルチャー | 趣味 / 推し活 / ライフスタイル コミュニティ |
| ミーム / 共有言語 | TikTok / X で循環している表現 |
| 影響力者 (KOL) | カテゴリー専門家、タレント、クリエイター |
| メディア生態系 | YouTube / Podcast / Newsletter / Discord 等の主要場所 |
| アーティスト / 作品 | 同時代の音楽 / 映像 / アートで響いているもの |

## 8. 機会領域 (White Space)

| 観点 | アプローチ |
|---|---|
| 未充足ニーズ | 顧客が我慢している / 諦めている領域 |
| 隣接カテゴリー | 越境で取り込める Job |
| サブターゲット | 既存ブランドが見落としている層 |
| 利用シーン拡張 | 既存ユーザーが想定外に使っているシーン |
| カテゴリー再定義 | カテゴリー自体の言い換えで広がる市場 |

## 9. リスク・地雷

| 観点 | 確認項目 |
|---|---|
| 過去の炎上事例 | カテゴリー / 競合 / 類似訴求 — 直近 3 年 |
| 規制リスク | 景表法 / 薬機法 / 食品表示 / 業界基準 |
| ブランドセーフティ | 隣接表現の取り扱い注意 (ジェンダー / ルッキズム / 健康 / 政治) |
| 文化的タブー | 季節 / 宗教 / 地域感情 |
| 競合被り | 同時期の競合キャンペーンとの重複 |
| 模倣リスク | 他社が真似しやすい構造になっていないか |

## 10. 仮説の種

プランナーへの引き継ぎカード。**結論ではなく "次に動ける問い" の形** で提示。

各仮説に以下を添える:

```
仮説 #N: <一文の仮説>
根拠:    <どのリサーチ結果から導いたか> (出典番号付き)
想定機会: <この仮説が当たった場合に開く可能性>
検証方法: <次にどう検証するか — 定量調査 / N1 インタビュー / A/B 等>
```
