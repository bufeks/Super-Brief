HYPERBOOST RUN ／ 日本市場・競合ディープダイブ FY26 ALPEN HYPERBOOST RUN ピッチ補足資料 — エージェンシーブリーフを超えて

本ドキュメントはブリーフ要約の補足として、日本ランニング市場のマクロ動向、競合の最新スペック・ポジショニング、ターゲットセグメントの解像度、アルペンの戦略文脈、HYPERBOOST RUNが取るべき差別化を整理したものです。

エグゼクティブサマリー

  • 日本ランニングシューズ市場は 4,200億円（'25）→ 6,600億円（'34、CAGR 4.96%） と中期成長軌道。世界でも有数の "プレミアム化" 進行マーケット。
  • 箱根駅伝 2026 で アディダスは2年連続シェア1位（35.7%）。エリート領域での強さを、エントリー層への "シャワー効果" として活用できる絶好機。
  • 女性ランニング人口は20年で最少水準（<200万人）に縮小する一方、 K-カルチャー連動の "ファッション・ランニング" が若年女性層で台頭。HYPERBOOST RUNの "Joy" 設計とフィットする数少ない潮流。
  • 競合のクッション帯は 255〜300g。247gのHYPERBOOST RUNはカテゴリ最軽量。"最大クッション × 最軽量 × アルペン限定" の三重差別化が勝ち筋。
  • アルペンは2026年4月に ALPEN RUN MEIJI PARK をオープン。社員1000人マラソン、年間300日ランイベント等、"運営側がランナー" の流通アセットは独自KOLとして使い倒すべき。

3. ターゲットセグメントの解像度

エントリーランナー全体像

人口

  • 男性 564万人／女性 175万人（'24）[2]
  • 男性40〜50代は実施率増加、全体は前年比微減 [2]

動機

  • 体型維持、生活習慣改善（健康文脈）
  • マラソン完走をゴール設定、コミュニティ加入で継続
  • "楽に走れて長く続くシューズ" を探索 [ブリーフ]

行動

  • クッション性を最重視（特に膝・腰の不安）
  • ハーフマラソン／リレマラソン参加が成長軸
  • "走る靴" を街でも履きたい [1][4]

女性層 — "縮小"と"再点火"の二重構造

  • 逆風：女性ランニング人口は 1998年の調査開始以来初めて200万人を下回り、20年で最低水準（笹川スポーツ財団）[2]。
  • 追い風：20代女性中心に K-カルチャー × ファッション × ランニング の若年層クルー文化が拡大。文化服装学院ランニング部 × 080TOKYO 等の事例が象徴 [1]。
  • 含意：HYPERBOOST RUNが狙うべき女性層は "競技完走志向のマス" ではなく、 "スタイルの一部としてラン" を始めた都市生活者。コミュニティ・SNS・撮りたくなる店頭装飾が必須。
提案ハイポセシス： KV／コピーは "走る人" ではなく "走り始める人 × 街に溶けるスニーカー" の文脈で構築。アルペンのウーマンズ専用 FTW WALL を作り、HOKA／NB 1080 から乗り換える層をピンポイントで奪う。

7. HYPERBOOST RUN ／ SWOT

STRENGTH

  • 箱根駅伝シェア1位ブランドの後光
  • EDGEからの新フォーム継承で説得力ある "弾む × 軽い"
  • 247gはクッション帯で実質最軽量
  • 3Dヒール構造でフィット感を物語化可能

WEAKNESS

  • ¥19,800は Novablast/Pegasus/NB1080 廉価帯より高い
  • HYPERBOOSTは新フランチャイズ＝ブランド認知ゼロから
  • EDGE が "プレミアム枠" を取った直後で価値の二段階構築が必要
  • 女性向けデザイン／カラー展開の訴求は未定

OPPORTUNITY

  • K-カルチャー連動 若年女性クルー文化が拡大中
  • ハーフマラソン参加が成長軸（+3pt）
  • アルペン明治公園 flagship との合流タイミング
  • HOKA Clifton 10 の重量増／NB 1080 の重量増は乗り換え好機
  • 9-10月の北海道マラソン／秋大会群でのサステイン

THREAT

  • Novablast 5 が "ベストバリュー" 評で同価格帯に刺さる
  • On / HOKA の街履き訴求が女性層に浸透

  • 夏ローンチの暑熱／イベントオペレーション負荷
